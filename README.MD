# Intro

CRUD nodejs app with authentication using JWT is sent via http-only cookie. The server will be available at `http://localhost:8081` 

## setup 

1. **Install the dependencies:**

```sh
yarn install
```

2. **Build the project:**

```sh
yarn run build
```

3. **docker:**

```sh
docker compose up -d
```

```sh
yarn run dev
```

## Testing

Basic the Jest test suite with:

```sh
yarn run test
``
